Update:

        Inventory:
        • Redesigned the display of stickers on skins and patches on agents. This excludes repeated requests for prices for the same stickers. Also, if you have floats loaded in your inventory, we request prices only from our API, completely excluding requests to Steam
        • Clicking on "Select Items" and "Select All" will automatically include and summarize the prices of the SIH Steam provider. But only if all providers have previously been disabled
        • "Simplified" / "Compact" inventory display mode will now be enabled by default
        • The price provider selection button has been duplicated near the sorts (in the future, the button under the inventory will be removed)
        • Using sorting by price will automatically switch on the "SIH Steam" price provider and sort by it. But only if no other price provider is currently enabled
        • Fixed a bug where after changing currencies and then trying to select multiple items in your inventory, the total amount was displayed incorrectly
        • Fixed a bug where floats were not displayed if all price providers were turned off
        Deal History:
        • Fixed a bug with incorrectly generated item link
        • After parsing, the deal history graph data will be saved again and will not be cleared on logout.
        Trades page:
        • Added ability to sort trades by 3 parameters.
        • Added possibility to download prices for each trade separately
        • Modified preloaders
        Item page:
        • Added link to csgostash for all Base Grade Container items (cases, capsules etc.) where you can view detailed information about container contents
        Globally:
        • Added direct purchase price for SkinsMonkey
